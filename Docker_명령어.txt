Docker는 기본적으로 리눅스에서만 실행된다.
Window는 버쳐박스 필요


docker version
docker info
docker --help (도커 명령어 확인)
docker images
docker ps (실행중인 컨테이너)
docker ps -a (실행했던 컨테이너)
docker run hello-world
docker-machine.exe ls

docker container run <docker-image-name> <command>
ex) docker container run ubuntu:latest /bin/echo 'Hello world'
